# Telegram debug checklist

- Confirmar bot/ambiente correto: teste ou produção.
- Registrar timestamp do envio.
- Registrar payload enviado ao bot.
- Confirmar se houve resposta em até 10 segundos.
- Capturar resposta exata retornada ao usuário.
- Verificar HTML/markup quebrado.
- Verificar se o erro foi silencioso ou explícito.
