# Common failures

## Silent failure
UI mostra resposta genérica, logs mostram exceção real.

## Timeout
Usuário não recebe resposta; backend bloqueado ou chamada externa demorando demais.

## Formatting issue
Lógica roda, mas HTML/markup da mensagem fica inválido.

## Query failure
Erro em ORM, tipagem, filtro ou serialização de resultado.
